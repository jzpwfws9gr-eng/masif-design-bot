# نسخة معدلة اعتماد فهد V70

هذه النسخة مبنية على V69 وتجمع كل التعديلات المعتمدة الأخيرة.

## أهم التغييرات
- تنبيهات المباراة كل 15 ثانية.
- تنبيهات المباراة مفعلة تلقائيًا للجميع، واللي يبي يوقفها يضغط زر الإيقاف.
- كل رسالة تنبيه تحتوي زر: ❌ إيقاف التنبيهات.
- لا يعيد إرسال أحداث قديمة بعد إعادة تشغيل البوت أو رفع نسخة جديدة؛ يبني خط أساس فقط ثم يرسل الجديد.
- مباشر الآن بصيغة مختصرة من ESPN مع الرموز: 🔴 مباشر، ⏳ لم تبدأ، ✅ انتهت.
- لوحة البطولة تُجبر تحديث البيانات قبل العرض، وتحسب المباريات المباشرة ضمن التفصيل والأهداف.
- إذا كانت لوحة البطولة مفتوحة، يحاول تحديثها دوريًا مع دورة تنبيهات المباراة.
- كيف يتأهل: عند وجود مباراة قائمة في مجموعة المنتخب، يعرض حسابًا مباشرًا مؤقتًا مرتبطًا بمباريات المجموعة كلها، وليس مباراة المنتخب فقط.
- دور الـ32: لا يثبت اسم منتخب إلا إذا ضمن الخانة 1000٪، ويعرض المتوقع الحالي تحت الخانات غير المحسومة.
