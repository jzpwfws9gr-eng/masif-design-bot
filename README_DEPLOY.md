# Masif Old Improved V60.5 — Direct Old Pull

هذه النسخة ترجع طريقة السحب القديمة من ملف `ممتتتاز .py`:

- مباشر الآن: أزرار + تفاصيل عند الضغط.
- النتائج: سحب مباشر عند الطلب مثل القديم.
- الهدافين: سحب مباشر من ESPN عند الطلب.
- المتأهلون والمغادرون: من دوال البوت القديمة مباشرة.
- تم إلغاء احتمالات المغادرة / سيناريوهات الخروج.
- `updater.py` صار نبض حالة فقط حتى لا يعلق في 8% أو مراحل ESPN.

## Railway Start Command

```bash
sh -c "python updater.py & python bot.py"
```

أو يمكن تشغيل البوت فقط:

```bash
python bot.py
```
