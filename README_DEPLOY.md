# نسخة V64 — إصلاح تنبيهات الأهداف

هذه النسخة مبنية على V63 وتصلح سبب عدم وصول إشعار الهدف رغم تفعيل التنبيهات قبل الهدف.

التعديلات:
- عند تفعيل 🔔 تنبيهات الأهداف، يبني البوت خط أساس مباشرة للنتائج الحالية بدل انتظار أول دورة.
- تنبيهات الأهداف تستخدم ذاكرة مستقلة ولا تعتمد على تحديث 📺 مباشر الآن.
- إذا JobQueue غير متوفر في Railway، توجد حلقة احتياط تشغل فحص التنبيهات كل 45 ثانية.
- أول نتيجة موجودة تعتبر خط أساس ولا ترسل أهداف قديمة.
- أي هدف جديد بعد الخط الأساس يرسل للمشتركين.

أوامر مساعدة:
- /تنبيهات_الأهداف — فتح تفعيل/إيقاف تنبيهات الأهداف.
- /حالة_تنبيهات_الأهداف — فحص حالة التنبيهات وعدد المشتركين وآخر تشغيل.
- /اختبار_تنبيه_هدف — إرسال رسالة اختبار للتأكد أن التنبيه يوصل.

باقي تعديلات V62/V63 مستمرة:

- زر دور الـ32 بالقائمة الرئيسية.
