الأساس: ملف ممتاز الشغال.

التعديل:
- بدون كاش جديد.
- بدون انتظار تحديث.
- بدون سحب ESPN بالخلفية.
- مباشر الآن / النتائج / الهدافين بنفس طريقة ملف ممتاز عند الضغط.
- حذف احتمالات المغادرة وسيناريوهات الخروج من الواجهة.
- أفضل الثوالث مع توضيح أفضل 8 من أصل 12.

Start Command المفضل للاختبار:
```bash
python bot.py
```

وإذا أردت إبقاء النبض:
```bash
sh -c "python updater.py & python bot.py"
```
