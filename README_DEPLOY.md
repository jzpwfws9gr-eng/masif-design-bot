# نسخة معدلة اعتماد فهد V67

هذه النسخة مبنية على V66 وتضيف تنبيهات المباراة التلقائية.

## التغييرات
- الحفاظ على حسبة دور الـ32 100٪ من V66.
- تنبيهات المباراة مفعلة تلقائيًا لكل المستخدمين المحفوظين في users.json.
- المستخدم يستطيع إيقاف التنبيهات من زر: 🔔 تنبيهات المباراة.
- تنبيهات المباراة تشمل:
  - بداية المباراة
  - نهاية الشوط الأول / بين الشوطين
  - بداية الشوط الثاني
  - نهاية المباراة
  - هدف جديد
  - إلغاء هدف
- آخر كل تنبيه يحتوي طريقة الإيقاف.
- لا يتم ذكر VAR عند إلغاء الهدف؛ فقط: تم إلغاء الهدف.

## أوامر مفيدة

ارفع الملفات على Railway بدل النسخة السابقة في نفس مشروع البوت القديم.
